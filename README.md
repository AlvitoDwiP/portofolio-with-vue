# Personal Portfolio Foundation

Fondasi awal portfolio personal dengan Vue 3 dan Vite. Setup ini sengaja dibuat minimal tetapi siap dikembangkan untuk kebutuhan one page portfolio dengan halaman detail project.

## Stack

- Vue 3
- Vite
- Vue Router
- Tailwind CSS
- VueUse Motion
- Lucide Icons
- @vueuse/head
- ESLint
- Prettier

## Available Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run format`

## Structure

```text
src/
├── assets/
├── components/
│   ├── layout/
│   ├── project/
│   ├── sections/
│   └── ui/
├── composables/

```
